Mixins

Introduction to Mixins

Mixins help us to create reusable blocks of code that can be applied in a number of situations. They improve efficiency and help insure consistency. Any code written inside of the mixin will be inserted when the mixin is called.

A basic mixin:

@mixin greenText { 
  color: green;
}

h2 { 
  font-size: 2rem;
  @include greenText;
}

Output:

h2 { 
  font-size: 2rem;
  color: green;
}

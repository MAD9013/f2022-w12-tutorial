Sass Basics

Introduction to Sass

Sass is a CSS pre-processor that allows us to write code more quickly and efficiently through improved organization, functionality, and code reusability.

Basic Commands

Single File Compile

sass ./sass/main.scss ./css/main.css

This will reference a single file and then will compile it into another referenced file. The first file reference is the source and the second referenced file is the destination.

sass ./{folder}/{entry-file} ./{output-folder}/{output-file}

Watched Folder Compile

sass --watch ./sass:./css

By using sass ./sass:./css we instead reference the entire folder and then compile the contents to a new folder. Adding a --watch flag will continue to run the Sass command and will recompile any time an altered file is saved. You can then end the watch command with control + c

sass --watch ./{watched-folder}:./{output-folder}

Compiled Style

sass --watch ./sass:./css --style compressed

We can modify the output by adding a --style flag. By default it will be expanded, which outputs easily readable CSS. If you put --style compressed it will compress the CSS down to one line. This should be used for final live version of your site to help reduce file size
